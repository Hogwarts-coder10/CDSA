# CDSA

Core Data Structures library written in C.

## Features

- Generic dynamic vector
- int-based LinkedList
- Generic Stack
- KString (Dynamic String Implementation)
- HashMap (Generic)
- RingBuffer (Circular Queue) (Generic)
- Queue
- Deque
- SkipList (for sorted sets)
- ART (Adaptive Radix Tree) (Supported upto Node4 and Node16)
- Modular architecture
- CMake build system
- Valgrind-tested memory safety

## Build

```bash
cmake -B build
cmake --build build
